Displays the content you want only the device you want, movil, tablet or only web.

== Description ==
With this plugin you can present the Widget on devices that you just want to show, ie if you only want to display the Widget on the web can set that only appears there. 


<h3>You can set it as follows: </h3>
<ul>
<li>If you want the widget to be displayed in a single <u>tablet  </u>. </li>
<li>If you want the widget to display only <u> mobile <u>phone </u>. </li>
<li>If you want the widget is displayed on the <u>web only </u>. </li>
</ul>
You can place a HTML text for display devices you want.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'WP Widget Devices'
3. Activate WP Widget Devices from your Plugins page.
4. Visit 'Appearance > Widget > 'WP Widget Devices'.
